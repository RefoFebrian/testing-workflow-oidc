Web DateTime Picker Today

Enhance date and datetime fields with quick-actions

Overview

This module simplifies the Odoo 18 date and datetime picker by introducing a smart contextual button at the bottom of the widget:

  • Today Button: Displayed on Date pickers. One-click selection for today's date.
  • Now Button: Displayed on Datetime pickers. One-click selection to set today's date and the current time (auto-aligned to Odoo's rounding interval configuration).

Key Features

Context-Aware Text

Shows "Today" for dates and "Now" for datetimes.

Time-Rounding Support

Automatically aligns seconds and minutes to the closest configuration interval (e.g., 5 min interval).

12/24H Format Ready

Seamlessly updates AM/PM and standard 12/24 hour display elements in the user's localized timezone.

Premium Integration

Integrates into standard footer actions alongside Close and Apply buttons with a responsive grid layout.
